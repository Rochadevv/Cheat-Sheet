FOUR COMMON CSS VISUAL PROPERTIES

CSS provides four common properties to alter the visual weight/emphasis of text.

font- style: Used to turn italic text on or off. Possible values are as follows (you'll rarely use this, unless you want to turn some italic styling off for some reason):

Propertie Value Description
font- style normal Sets the text to normal font(turns existing italics off).
font- style italic Sets the text to use the italic version of the font, if variable; if not , it will simulate italics with oblique instead.
font- style oblique Sets the text to use a simulated version of an italic font, created by slanting the normal version.

font-weight : Sets how bold the text is. This has many values available in case you have many font variants available (such as -light, -normal, -bold, -extrabold, -black, etc.), but realistically you'll rarely use any of them except for normal and bold:

Propertie Value Description
font-weight normal, bold Normal and bold font weight.
font-weight lighter, bolder Sets the current element's boldness to be one step lighter or heavier than its parent element's boldness.
font-weight 100â€“900 Numeric boldness values that provide finer grained control than the above keywords, if needed.

text-transform : Allows you to set your font to be transformed. Values include:

Propertie Value Description
text-transform none Prevents any transformation.
text-transform uppercase Transforms all text to capitals.
text-transform lowercase Transforms all text to lower case.
text-transform capitalize Transforms all words to have the first letter capitalized.
text-transform full-width Transforms all glyphs to be written inside a fixed-width square, similar to a monospace font, allowing aligning of, e.g., Latin characters along with Asian language glyphs (like Chinese, Japanese, Korean).

text-decoration: Sets/unsets text decorations on fonts (you'll mainly use this to unset the default underline on links when styling them). Available values are:

Propertie Value Description
text-decoration none Unsets any text decorations already present .
text-decoration underline Underline the text.
text-decoration overline Gives the text on overline.
text-decoration line-through Puts a strikethrough over the text.

You should note that text-decoration can accept multiple values at once if you want to add multiple decorations simultaneously, for example, text-decoration: underline overline. Also note that text-decoration is a shorthand property for text-decoration-line, text-decoration-style, and text-decoration-color. You can use combinations of these property values to create interesting effects, for example: text-decoration: line-through red wavy.
